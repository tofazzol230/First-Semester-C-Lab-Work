## 📋 Lab Topics Overview

### 1️⃣ Arrays (`bsse_1740_array/`)
Basic array operations including:
- Array traversal & manipulation
- Searching & sorting
- Multi-dimensional arrays

### 2️⃣ Advanced Arrays (`bsse_1740_adv_arr/`)
Complex array problems:
- One Edit Away (String similarity)
- Palindrome checking
- Parentheses validation
- Rotation algorithms

### 3️⃣ Pointers (`bsse_1740_pointer/`)
Pointer fundamentals:
- String operations (copy, compare, length) using pointers
- Merge sort with pointers
- Min/Max finding with pointers

### 4️⃣ Adjacency List & Pointers (`bsse_1740_adj_list_ptr/`)
Graph representation:
- Adjacency list implementation
- Adjacency matrix implementation
- String sorting with pointers

### 5️⃣ Recursion (`bsse_1740_recursion/`)
Recursive algorithms:
- Classic recursion problems
- Mathematical computations
- Divide and conquer

### 6️⃣ Strings (`bsse_1740_string/`)
String manipulation:
- String operations
- Character processing

### 7️⃣ String Regex (`bsse_1740_string_regex/`)
Pattern matching:
- Regular expression operations
- Pattern validation

### 8️⃣ Pattern Printing (`bsse_1740_pattern_printing/`)
Visual patterns:
- Number patterns
- Star patterns
- Pyramid structures

### 9️⃣ File Handling (`bsse_1740_file/`)
File I/O operations:
- Reading from files
- Writing to files
- File manipulation

### 🔟 Graph Algorithms (`bsse_1740_graph/`)
Graph theory:
- Graph traversal (BFS/DFS)
- Graph representation

### 1️⃣1️⃣ Struct & BMP Processing (`bsse_1740_struct_lena_bmp/`)
Structures & image processing:
- BMP file format handling
- Grayscale conversion
- Struct usage in C

## 🛠️ How to Compile & Run

### Compile any C file:
```bash
gcc filename.c -o output
./output
```

### Example:
```bash
cd bsse_1740_array
gcc bsse_1740_array_1-1.c -o array1
./array1
```
